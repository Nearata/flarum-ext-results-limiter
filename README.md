# Results Limiter

A [Flarum](http://flarum.org) extension. No description.

## Install

```sh
composer require nearata/flarum-ext-results-limiter:"*"
```

## Update

```sh
composer update nearata/flarum-ext-results-limiter:"*"
php flarum migrate
php flarum cache:clear
```
